CostingOptions

public struct CostingOptions : Codable, @unchecked Sendable

Configuration options for route costing and avoidance preferences.

Use CostingOptions to specify routing preferences such as avoiding tolls, highways, or ferries, and to control various routing parameters including vehicle specifications, road preferences, and cost factors.

Usage

let options = CostingOptions(
    avoidTolls: true,
    avoidHighways: false,
    shortcut: true,
    maneuverPenalty: 5.0,
    countryChangingPenalty: 600.0
)

All properties are optional, allowing you to specify only the preferences you need. Different costing models support different subsets of these options.

Avoidance Options
