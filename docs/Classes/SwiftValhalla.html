  • configure(serverURL:)

    Configure the shared SwiftValhalla instance with a server URL string.

    This parses the provided serverURL string into a URL and stores it on SwiftValhalla.shared.serverURL for use throughout the library.

    Throws

    An NSError with domain "SwiftValhalla" and code 1001 when the provided string cannot be converted to a valid URL.

    Note

    Call once early in your app’s lifecycle (for example in AppDelegate or during startup) before performing network operations.
