Filter

public enum Filter : String, Codable, CaseIterable, Sendable

Represents all available filter keys that can be applied when working with Valhalla routing and map-matching data.

Each case corresponds to a specific field in the Valhalla API, grouped by category (edge, node, admin, matched, etc.). The raw value matches the exact key string expected by Valhalla.

Conformance to Codable makes it easy to serialize and deserialize filters, and CaseIterable allows iteration over all supported filters.
